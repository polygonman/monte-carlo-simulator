# Monte Carlo Simulator

A retirement planning simulator using Monte Carlo methods, built with React + Vite.

## Features
- Simulates 500–5,000 random market scenarios using Box-Muller normal distribution
- Shows P10/P25/P50/P75/P90 wealth paths over time
- Inflation-adjusted withdrawals and contributions
- Success rate calculation (% of simulations ending with positive balance)
- Interactive charts with recharts

## Stack
- React 19 + Vite 8
- recharts for charts
- lucide-react for icons
- Tailwind CSS v4

## Run Locally
```bash
npm install
npm run dev
```
